# Sistemas Robóticos — Material de Estudo

Repositório de estudo para a UC de Sistemas Robóticos (Mestrado em Eng. Eletrotécnica — ISEC).

## Como usar

Usa a barra lateral para navegar entre fichas de estudo, análise de exames e enunciados.

Começa por:
- [Frequência dos Temas](frequencia_dos_temas.md) — o que sai mais vezes
- [Ficha 01 — Denavit-Hartenberg](Fichas_Estudo/01_Denavit_Hartenberg.md) — tema presente em todos os exames

## Estrutura

| Pasta | Conteúdo |
|---|---|
| `Fichas_Estudo/` | Fichas por tema (10 temas) |
| `Exames/` | Enunciados extraídos |
| `Resolucoes/` | Resoluções de colegas (⚠️ não validadas) |
| `00_ORIGINAIS/` | Ficheiros originais intactos |
