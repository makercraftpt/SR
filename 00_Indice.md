**Para o exame:**
1. Domina DH + Cinemática Direta + Inversa (Q1+Q2+Q3 = 45–50% da nota)
2. Pratica as matrizes de transformação com exemplos concretos (Q4+Q5 = 25%)
3. Revê o Jacobiano para velocidades (Q6 = 10%)
4. Lê as fichas 07–10 para a pergunta teórica (Q7/Q8 = 10–20%)
